BigBang Theme
==============================================
Crazy Brackets theme for you.


### Supported Languages:
- **HTML,XML**
- **CSS, SCSS, SASS**
- **JavaScript**
- **Php**

Installation
---

1. Open Brackets
2. Open the Extension Manager
3. Switch to "Themes" tab
4. Search for "BigBang"
5. Click "Install"
